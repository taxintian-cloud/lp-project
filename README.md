# 個人事業主向けシンプルLP

## 🚀 コンセプト
SNSはがんばって更新しているのに、予約は増えない――  
そんな個人事業主・ひとり経営者のために作った、**シンプルで迷わず問い合わせまで誘導できるLP**です。  

- 派手な集客より、**必要な人が迷わず行動できる仕組み**を重視  
- 難しい専門用語は使わず、直感的に理解できるデザイン  
- モバイル・PC対応のレスポンシブ設計  

---

## 🎯 今回の取り組み（2か月目の挑戦）
- HTML/CSSの基礎をベースに、**レスポンシブでシックなLP**を自作  
- 行間・余白・ボタンの呼吸感など、**見た目のリズムを徹底的に調整**  
- 強調文やCTAボタンを戦略的に配置し、**ユーザー行動を誘導する設計**を学習  
- GitHubでの公開も視野に入れ、**実際にアクセスできるLP制作**に挑戦  

---

## ✨ 特徴
- シンプルで整ったデザイン  
- スクロールや余白、文字サイズの階調を意識  
- CTAボタンはクリックしやすく、強調文章とのリズムも調整済み  
- 小さな改善を積み重ねることで、**見た目と導線の両立**を実現  

---

## 📂 ファイル構成
lp-project/

## ⚡ 利用方法
1. このリポジトリをクローンまたはダウンロード  
2. `index.html` をブラウザで開く  
3. 必要に応じてテキストやリンクをカスタマイズ
